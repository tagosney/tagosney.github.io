My Story

In 1995, I wrote my first technical manual for a software product that would track business processes. That project piqued my interest in pursuing a career in technical writing. I continued to work for several companies that helped my career grow.

The documentation companies I worked for had many clients in the medical, waste disposal, oilfield, and software industries. After writing software documentation, I gained an interest in software development.

In 2008, I started my journey to work on an associate’s degree in computer programming. I continued working full-time as a technical writer while taking college courses and graduated in December 2011. Within a month, I changed careers and started working as a software developer.

In 2016, I decided that I wanted to pursue a bachelor’s degree in computer information systems. So, I continued working as a software developer while taking college courses and graduated in December 2018. My skills range from working with front end to back end of web applications using .Net Framework and .Net Core.
